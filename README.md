# 📚 智能教案写作工具

一个现代化的在线教案生成工具，帮助教师快速创建结构化的教学计划。

## ✨ 主要功能

- **快速生成教案**: 填写基本信息后，自动生成完整的教案框架
- **智能结构化**: 包含导入、讲授、实践、总结等完整教学环节
- **差异化教学**: 考虑不同学生层次的教学方法
- **多格式支持**: 支持网页查看、复制、下载
- **响应式设计**: 支持桌面和移动设备

## 🚀 快速开始

1. 克隆或下载本仓库
2. 在浏览器中打开 `index.html` 文件
3. 填写教案基本信息
4. 点击"生成教案"按钮
5. 查看、复制或下载生成的教案

## 📋 使用说明

### 输入信息

- **学科**: 填写教学学科（数学、语文、英语等）
- **年级**: 选择学生所在年级
- **单元/章节**: 填写本课所属的单元或章节
- **课题**: 输入本次课的课题名称
- **课时**: 选择本课的时长
- **学生人数**: 填写班级学生数量
- **教学目标**: 描述知识、能力、情感等方面的目标
- **重点难点**: 列出本课的重难点

### 生成选项

- **生成教案**: 根据输入信息自动生成教案
- **下载教案**: 将教案内容下载为文本文件
- **复制内容**: 一键复制教案内容到剪贴板
- **重新生成**: 清空表单重新开始

## 💡 教案内容包括

- 基本信息
- 教学目标
- 教学重点和难点
- 教学方法
- 教学资源
- 教学过程设计（导入、讲授、实践、总结）
- 课堂纪律管理
- 作业与拓展
- 教学评价
- 教学反思
- 差异化教学考虑

## 🔧 自定义

你可以编辑 `script.js` 文件中的 `generateLesson()` 函数来自定义教案模板和内容。

## 🎯 使用建议

1. **基础框架**: 此工具生成的教案是基础框架，建议根据实际教学情况进行详细补充
2. **个性化调整**: 根据学生特点和学校要求进行修改和完善
3. **分层教学**: 充分利用工具中的差异化教学部分，照顾不同学生群体
4. **教学反思**: 生成教案后，课堂实践中不断总结和改进
